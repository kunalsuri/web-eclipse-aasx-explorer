# Pre-completion checklist
- [ ] Running in a session that did NOT implement the change
- [ ] Diff scope + authorizing spec pinned (no spec ⇒ blocker finding #1)
- [ ] `REVIEW_<work-id>.md` created from `ai/lab/reviews/REVIEW_TEMPLATE.md`
- [ ] Every check backed by where-you-looked evidence, not assertion
- [ ] Spec's suites re-run in this session, not trusting the implementer's report
- [ ] Findings severity-ranked; verdict set (`approve` / `request-changes`)
- [ ] "What the human should double-check" filled; review tagged `[inferred]`
- [ ] WORKLOG Review column linked; row Status set to `in-review`
